Вход сотрудника

Вход для замерщика: логин: ruslan / rifat / vildan / vitalik код выдаёт администратор

Доступ только для замерщиков, конструктора, менеджера и администратора.

Введите короткий логин: ruslan, rifat, vildan, vitalik или полный email.

Доступ выдаёт администратор. Для работы используйте выданный логин и код.

Установить на телефон

iPhone Safari → Поделиться → На экран «Домой»
Android Chrome → ⋮ → Добавить на главный экран

Поделиться с замерщиком

Выберите логин, скопируйте текст и отправьте его в WhatsApp или Telegram.
